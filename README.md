
## Prérequis

 - Java 6 ou 7 [Download](http://www.oracle.com/technetwork/java/javase/downloads/index.html)
 - Java Cryptography Extension [Download](http://www.oracle.com/technetwork/java/javase/downloads/index.html)
 - L'application web [anon-standalone.jar](anon-standalone.jar).

## Outils suggérés

 - [Subgraph Vega](http://subgraph.com/products.html) / [Burp Suite](http://portswigger.net/burp/) / [WebScarab](https://www.owasp.org/index.php/Category:OWASP_WebScarab_Project)
 - [Mozilla Firefox](http://www.mozilla.org/firefox/new/) (proxy facile à configuré)
 - [Wireshark](http://www.wireshark.org/)
 - [Python](http://www.python.org/download/) / [Ruby](http://www.ruby-lang.org/) / [Perl](http://www.perl.org/)

## Mode de fonctionnement

 - Exécuter la commande suivante: <code>java -jar anon-standalone.jar</code>

 - Ouvrir un navigateur web à l'addresse: [http://localhost:9999](http://localhost:9999)

 - Lire [INSTRUCTIONS.md](INSTRUCTIONS.md)
